Documentation

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON Object holding data about all the movies
Return data about a single movie by title to the user /movies[Title] GET None A JSON Object holding data about a single movie, containing title, releaseDate, language, etc..is returned.
{ "title": "F9",
"releaseDate": "June 25, 2021",
"language": "English",
}
Return data about a genre (description) by movie title  /movies[Title]/genre GET None A text message indicating general description of the movie. (e.g., “Thriller”)
Return data about a director by name /directors[Name] GET None A JSON Object holding data about a director, containing (bio, birth year, death year
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{ "name": "User1",
"email": "user1@xyz.com",
"movies_List": ["F9", "Palmer"]
} 		A JSON object holding data about the user that was added, including an ID:
{ "id": 1,
"name": "User1",
"email": "user1@xyz.com",
"movies_List": ["F9", "Palmer"]
}
Allow users to update their user info (username) /users/[Name]/[Email] PUT None A text message indicating the user that email was updated.
Allow users to add a movie to their list of favorites  /users/[Name]/[MovieTitle] POST None A text message indicating the favorite movie has been added to the list.
Allow users to remove a movie from their list of favorites  /users/[Name]/[MovieTitle] DELETE None A text message indicating the favorite movie has been removed from the list.
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users[ID] DELETE None A text message indicating whether the user details has been removed.
